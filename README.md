## Installation
Firstly, you need clone this project to local computer: 
```bash
git clone https://github.com/baohuy2209/ai-taking-app.git
```

Install my-project with pnpm

```bash
cd ai-taking-app
pnpm install
```
Then, run project with command 
```bash
pnpm run dev
```
